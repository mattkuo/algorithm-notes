# Primitive Types

## Sizes of Types

In Java, sizes of data types are machine independent. A word in the JVM is 32-bits.

| Data Type        | Bytes | Bits |
| ---------------- | ----- | ---- |
| boolean          | -     | 1    |
| byte             | 1     | 8    |
| short            | 2     | 16   |
| char             | 2     | 16   |
| int              | 4     | 32   |
| float (decimal)  | 4     | 32   |
| long             | 8     | 64   |
| double (decimal) | 8     | 64   |
